# Smart Hospital System

A comprehensive web-based Hospital Management System designed to streamline healthcare facility operations. This system integrates multiple modules to manage patient flow from reception to diagnosis, treatment, and medication dispensing.

## Features

- **Reception**: Patient registration, appointment scheduling, and queue management.
- **Doctor Interface**: View patient history, add diagnosis, and prescribe medications or lab tests.
- **Lab Management**: Process lab requests and update test results.
- **Pharmacy**: View prescriptions and manage medicine dispensing.
- **Patient View**: Interface for patients/displays to see queue status or basic info.
- **Admin Dashboard**: System maintenance and settings.

## Technologies Used

- **Frontend**: HTML5, CSS3, Bootstrap 5, jQuery
- **Backend**: PHP
- **Database**: MySQL

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AhmedJawad99/Smart-Hospital-System.git
   ```
2. Import the `database_setup.sql` into your MySQL database.
3. Configure the database connection in `db.php`.
4. Run the application on your XAMPP/WAMP server.
